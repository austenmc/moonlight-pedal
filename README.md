# Moonlight Through the Pines Pedal

Arduino guitar pedal.

## Training
* Looper / Clear (tap+hold)
* Looper / Start/Stop
* Tuner (tap+hold)
* Preset+
* Preset-
* Clone Drive
  * Gain
  * Tone
  * Level
  * Toggle
* 808OD
  * Gain
  * Tone
  * Level
  * Toggle
* EchoreX HD
  * Length of Swell
  * Tone
  * Volume
  * Toggle
* Digital Reverb
  * Reverb
  * Tone
  * Length
  * Toggle
